## 1. Technical SEO Audit
- [ ] **XML Sitemap:** Ensure your `sitemap.xml` is dynamic, clean (no 404s/redirects), and submitted to Google Search Console.
- [ ] **Robots.txt:** Verify that critical rendering resources (CSS/JS) are not blocked.
- [ ] **Indexability:** Check for accidental `noindex` tags on money pages.
- [ ] **Crawl Errors:** Review GSC for 4xx and 5xx errors and set up proper 301 redirects.
- [ ] **Schema Markup:** Validate JSON-LD structured data (FAQ, Article, LocalBusiness) using the Rich Results Test.
- [ ] **HTTPS/SSL:** Ensure all HTTP variations 301 redirect to the secure HTTPS version.

## 2. Performance & Server Optimization
Speed is a direct ranking factor. Fast sites equal better user experience and higher conversions.

- [ ] **TTFB (Time to First Byte):** Aim for a TTFB of under 200ms. This is heavily dependent on your server quality.
- [ ] **LCP (Largest Contentful Paint):** Target < 2.5 seconds.
- [ ] **Image Optimization:** Serve images in next-gen formats (WebP/AVIF) and implement lazy loading.
- [ ] **Minification:** Minify CSS, JS, and HTML files.
- [ ] **Caching Strategy:** Ensure proper browser caching and server-side caching (e.g., Redis, Memcached) are active.

## 3. On-Page & Content Strategy
- [ ] **Search Intent:** Does the page format (listicle, guide, tool) match what the user actually wants?
- [ ] **Header Tags (H1, H2, H3):** Ensure logical hierarchy. Only one H1 per page.
- [ ] **Internal Linking:** Use descriptive anchor text to distribute page authority to your most important pages (Silo structure).
- [ ] **Orphan Pages:** Identify and link out to pages that currently have no internal links pointing to them.
- [ ] **Meta Descriptions:** Keep them compelling and within 150-160 characters to maximize CTR in search results.
